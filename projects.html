>LOTR_Bot - Current Project

LOTR_Bot is a fun side project I have been working on that is a bot for the Discord application. The functionality of the bot is simple: It analyzes messages in a Discord server and searches for keywords or key phrases. Should a word or phrase be used, the bot will provide a quote from The Lord of the Rings. I have been running the bot from a Raspberry Pi in headless mode, using SSH to update the bot as I develop it.

Languages used include Javascript and Bash scripting.

Knight's Errand - Inactive<

Previous Next

This was my first and only major foray into game development. I have developed small sample projects before, but this undertaking was on a scale I have yet to repeat and was a lot of fun but very time consuming. The game was an "Infinite Runner" and consisted of a Knight character being controlled by the player running on an infinite, randomly generated path with obstacles. The player would control when the character would attack, roll, or jump over said obstacles. Development stopped due to life circumstances but I still retain the code and intend to return one day.

This game was developed in the Unreal 4 Engine using mostly their built in Blue Prints IDE with some C++ coding.
